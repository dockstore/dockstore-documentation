Reasoning: Sphinx does not support putting figures (images with captions) side-by-side. But, we want to break up our index page with some images so it isn't just a wall of text. If we want to avoid using full-size logos (which have different sizes and aren't very neat when placed side by side, and some of them do not have high-res images easily available), we can use square logos. But we also want to make it clear what logo represents what launch-with partner, we ought to make an invisible table. We're rendering it in this page and then screenshotting it as an image, because doing this sort of thing in RST is a bit finicky.

In GIMP, we can then set guides via Image>Guides>New Guide (by percent) at the following horizontal values:

  • 16.67%
  • 33.34%
  • 50%
  • 66.67%
  • 83.34%

Then, we can split along those guides.
